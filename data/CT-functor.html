Given two categories \(\mathcal{C}\) and \(\mathcal{D}\), a functor \(F : \mathcal{C} \to \mathcal{D}\) assigns an object (resp. a morphism) in \(\mathcal{D}\) to each object (resp. morphism) in \(\mathcal{C}\), such that
  • \(F(f) : F(X) \to F(Y)\) for each \(f : X \to Y\),
  • \(F(gf) = F(g) F(f)\),
  • \(F(\id_X) = \id_{F(X)}\).

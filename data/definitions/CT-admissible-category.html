A locally small category \(\mathcal{C}\) is \(\kappa\)-accessible, for a regular cardinal \(\kappa\), if
  • \(\mathcal{C}\) has \(\kappa\)-filtered colimits,
  • there exists a set of \(\kappa\)-compact objects that generates \(\mathcal{C}\) under \(\kappa\)-filtered colimits.
The category \(\mathcal{C}\) is accessible if it is \(\kappa\)-accessible for some \(\kappa\).

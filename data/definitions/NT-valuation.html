A valuation on a field \(K\) is a function \(\phi : K \to \RR_{\ge 0}\) satisfying
  • \(\phi(x) = 0\) if and only if \(x = 0\),
  • \(\phi(xy) = \phi(x) \phi(y)\) for \(x, y \in K\),
  • there exists \(C > 0\) such that \(\phi(x + y) \le C \max \{ \phi(x), \phi(y) \}\) for all \(x, y \in K\).
The smallest possible constant \(C\) is called the norm \(\norm{\phi}\) of \(\phi\). Note that \[ \norm{\phi} = \sup_{x : \phi(x) \le 1} \phi(1 + x) . \]

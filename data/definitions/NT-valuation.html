A valuation on a field \(K\) is a function \(\phi : K \to \RR_{\ge 0}\) satisfying
  • \(\phi(x) = 0\) if and only if \(x = 0\),
  • \(\phi(xy) = \phi(x) \phi(y)\) for \(x, y \in K\),
  • there exists a constant \(C > 0\) such that \(\phi(x + y) \le C \max \{ \phi(x), \phi(y) \}\) for all \(x, y \in K\).
The smallest possible constant \(C\), called the norm \(\norm{\phi}\) of \(\phi\), is given by \[ \norm{\phi} = \sup \{ \phi(1 + x) : x \in K \textup{ s.t. } \phi(x) \le 1 \} . \]

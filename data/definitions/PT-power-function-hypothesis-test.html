The power function of a hypothesis test with rejection region \(R\) is the function \(\beta : \Theta \to \RR\) on the parameter space \(\Theta\) given by \[ \beta(\theta) = \PP_\theta(X^{-1}(R)) . \] The hypothesis test is said to have size \(\alpha\) if \(\sup_{\theta \in \Theta_0} \beta(\theta) = \alpha\).
The hypothesis test is said to have level \(\alpha\) if \(\sup_{\theta \in \Theta_0} \beta(\theta) \le \alpha\).

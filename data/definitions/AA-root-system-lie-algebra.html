Let \(\mathfrak{g}\) be a complex semisimple Lie algebra, and \(\mathfrak{h}\) a Cartan subalgebra. A root of \(\mathfrak{g}\) (relative to \(\mathfrak{h}\)) is an element \(\alpha \in \mathfrak{h}^*\) such that \[ \mathfrak{g}_\alpha = \{ x \in \mathfrak{g} : [h, x] = \alpha(h) x \textup{ for all } h \in \mathfrak{h} \} \] is non-empty. The root system of \(\mathfrak{g}\) (relative to \(\mathfrak{h}\)) is the set \(\Phi\) of all roots. In particular, \(\Phi\) is a root system in \(\mathfrak{h}^*\).

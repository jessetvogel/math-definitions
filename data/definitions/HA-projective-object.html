Let \(\mathcal{A}\) be an abelian category. An object \(P\) of \(\mathcal{A}\) is called projective if the functor \[ \Hom_\mathcal{A}(P, -) : \mathcal{A} \to \textbf{Ab} \] is exact. Since it is already right exact for any object \(P\), this is equivalent to the condition that for all epimorphisms \(f : N \to M\) and morphisms \(g : P \to M\) there exists a morphism \(h : P \to N\) such that \(fh = g\).

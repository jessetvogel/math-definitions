Let \((\Omega, \mathcal{A}, \PP)\) be a probability space. A sequence of random variables \(X_i : \Omega \to (E_i, \mathcal{E}_i)\) for \(i = 1, 2, \ldots, n\) is (mutually) independent if \[ \PP(X_i \in A_i \textup{ for all } i = 1, 2, \ldots, n) = \prod_{i = 1}^{n} \PP(X_i \in A_i) \] for all \(A_i \in \mathcal{E}_i\).

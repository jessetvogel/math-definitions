A simplicial set is a functor \[ X : \Delta^\text{op} \to \textbf{Set} , \] where \(\Delta\) denotes the simplex category. Concretely, a simplicial set consists of a family of sets \(X_n\) and a map \(X_m \to X_n\) for each function \(f : [n] \to [m]\), which compose nicely.
The face maps are the maps \(d_{n, i} : X_n \to X_{n - 1}\) corresponding to the function \(\delta^{n, i} : [n - 1] \to [n]\) that misses \(i\).
The degeneracy maps are the maps \(s_{n, i} : X_n \to X_{n + 1}\) corresponding to the function \(\sigma^{n, i} : [n + 1] \to [n]\) that repeat \(i\).

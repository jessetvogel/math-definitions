Let \(R\) be a ring. An element \(a \in R\) is called a left zero-divisor if \(ab = 0\) for some \(b \ne 0\). Similarly, an element \(a \in R\) is called a right zero-divisor if \(ba = 0\) for some \(b \ne 0\). An element \(a \in R\) is called a zero-divisor if it is both a left zero-divisor and a right zero-divisor.

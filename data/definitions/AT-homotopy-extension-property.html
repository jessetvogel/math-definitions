Let \(X\) be a topological space and \(A \subset X\) a subspace. Then the pair \((X, A)\) has the homotopy extension property (HEP) if for any topological space \(Z\), continuous map \(f : X \to Z\) and homotopy \(F : A \times [0, 1] \to Z\) with \(F|_{A \times \{ 0 \}} = f|_A\), there exists a homotopy \(H : X \times [0, 1] \to Z\) with \(H|_{X \times \{ 0 \}} = f\) and \(H|_{A \times [0, 1]} = F\).

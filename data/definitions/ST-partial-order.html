A partial order on a set \(A\) is a binary relation \(\le\) on \(A\) such that
  • (reflexivity) \(x \le x\) for all \(x \in A\),
  • (anti-symmetry) if \(x \le y\) and \(y \le x\) then \(x = y\) for all \(x, y \in A\),
  • (transitivity) if \(x \le y\) and \(y \le z\) then \(x \le z\) for all \(x, y, z \in A\).
A partially ordered set is a set \(A\) together with a partial order \(\le\).

A metric space is a set \(X\) together with a metric \(d\) on \(X\), that is, a function \(d : X \times X \to \RR\) such that
  • (indiscernibility) \(d(x, y) = 0\) if and only if \(x = y\),
  • (symmetry) \(d(x, y) = d(y, x)\) for all \(x, y \in X\),
  • (triangle inequality) \(d(x, z) \le d(x, y) + d(y, z)\) for all \(x, y, z \in X\).

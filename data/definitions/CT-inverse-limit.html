Let \(\mathcal{C}\) be a category. An inverse system in \(\mathcal{C}\) consists of a directed set \(I\), an object \(X_i\) for each \(i \in I\), and a morphism \(f_{ij} : X_j \to X_i\) for all \(i \le j\), such that \(f_{ii}\) is the identity on \(X_i\) and \[ f_{ij} \circ f_{jk} = f_{ik} \quad \text{for all } i \le j \le k . \] An inverse limit of an inverse system is an object \(X = \varprojlim_{i \in I} X_i\) in \(\mathcal{C}\) with maps \(\pi_i : X \to X_i\) for each \(i \in I\) such that \(\pi_i = f_{ij} \circ \pi_j\) for all \(i \le j\), satisfying the universal property: for any other such object \(Y\) with maps \(p_i : Y \to Y_i\) there is a unique map \(h : Y \to X\) such that \(p_i = \pi_i \circ h\) for all \(i \in I\). Equivalently, an inverse system is a functor \(F : I^\text{op} \to \mathcal{C}\) for a directed set \(I\), and an inverse limit is a limit for \(F\).

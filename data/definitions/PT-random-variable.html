Let \((\Omega, \mathcal{A}, \PP)\) be a probability space. A random variable on \(\Omega\) is a measurable function \(X : \Omega \to \RR\), where \(\RR\) is equipped with the Borel \(\sigma\)-algebra.
More generally, if \((E, \mathcal{E})\) is a measurable space, then an (\(E\)-valued) random variable on \(\Omega\) is a measurable function \(X : \Omega \to E\).

A graph consists of a set \(V\) together with a set \(E \subset \{ \{ x, y \} : x, y \in V \textup{ and } x \ne y \}\).
A directed graph consists of a set \(V\) together with a set \(E \subset \{ (x, y) : x, y \in V \textup{ and } x \ne y \}\).
The elements of \(V\) are called vertices and the elements of \(E\) are called edges.

Let \(M\) be a smooth manifold, and \(E \to M\) a vector bundle on \(M\) with connection \(\nabla\). A section \(s : M \to E\) is called parallel if \(\nabla s = 0\).
The problem of parallel transport asks, given a point \(p \in M\) and vector \(v \in E_p\), if there exists (locally) a section \(s : U \to E\) such that \(\nabla s = 0\) and \(s(p) = v\), for some neighborhood \(U \subset M\) of \(p\). If it exists, this section is called the parallel transport of \(v\). It is a theorem that this can be done for all \(p\) and \(v\) if and only if the curvature \(F_\nabla\) vanishes.

Let \(X\) be a non-singular projective variety over an algebraically closed field \(k\). A complete linear system on \(X\) is a set \(|D|\) of effective Weil divisors linearly equivalent to a given divisor \(D \in \operatorname{Div}(X)\).
If \(\mathcal{L}\) is a line bundle over \(X\) corresponding to \(D\), there is a bijection \[ H^0(X, \mathcal{L}) / k^* \cong |D| \] which assigns to a non-zero section \(s\) of \(\mathcal{L}\) the divisor \(D + \operatorname{div}(s)\), giving \(|D|\) the structure of a projective vector space over \(k\). A linear system on \(X\) is a linear subspace of a complete linear system \(|D|\).

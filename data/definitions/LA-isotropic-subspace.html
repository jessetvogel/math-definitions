Let \((V, \omega)\) be a symplectic vector space. A linear subspace \(W \subset V\) is isotropic if \(W \subset W^\perp\), and coisotropic if \(W^\perp \subset W\).
Equivalently, \(W\) is isotropic if and only if \(\omega|_W = 0\). And \(W\) is coisotropic if and only if \(\omega\) restricts to a symplectic form on \(W/W^\perp\).

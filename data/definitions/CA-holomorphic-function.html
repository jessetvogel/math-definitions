A function \(f : U \to \CC\), with \(U \subset \CC\) an open subset, is holomorphic if it is complex differentiable for all \(z \in U\).
A function \(f : U \to \CC^n\), with \(U \subset \CC^m\) an open subset, is holomorphic if every component is holomorphic as a function in each variable separately, with the other variables fixed.
A map \(f : X \to Y\) between complex manifolds is holomorphic if for every point \(x \in X\), there is a chart \((U, \varphi)\) on \(X\) with \(x \in U\) and a chart \((V, \psi)\) on \(Y\) with \(f(U) \subset V\), such that the composition \[ \varphi(U) \xrightarrow{\varphi^{-1}} U \xrightarrow{f} V \xrightarrow{\psi} \psi(V) \subset \CC^n \] is holomorphic.

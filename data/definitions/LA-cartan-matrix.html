A generalized Cartan matrix is a square matrix \(A\) with integral entries \(a_{ij} \in \ZZ\) such that \[ a_{ii} = 2, \quad a_{ij} \le 0 \textup{ if } i \ne j, \quad a_{ij} = 0 \textup{ if and only if } a_{ji} = 0 . \] A generalized Cartan matrix \(A\) is symmetrizable if \(A = DS\) for some diagonal matrix \(D\) and symmetric matrix \(S\). If, moreover, \(D\) can be chosen with positive diagonal entries and \(S\) positive definite, then \(A\) is called a Cartan matrix.

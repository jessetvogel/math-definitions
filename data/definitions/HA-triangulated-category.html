Let \(\mathcal{D}\) be an additive category. The structure of a triangulated category on \(\mathcal{D}\) is given by an additive equivalence \[ [1] : \mathcal{D} \to \mathcal{D} \qquad \text{(the shift functor)} \] and a set of distinguished triangles \[ A \to B \to C \to A[1] \] such that
    • Any \(A \to A \to 0 \to A[1]\) is distinguished.
    • Any triangle isomorphic to a distinguished triangle is distinguished.
    • Any morphism \(f : A \to B\) can be completed to a distinguished triangle \(A \overset{f}{\to} B \to C \to A[1]\).
  2. A triangle \[ A \xrightarrow{f} B \xrightarrow{g} C \xrightarrow{h} A[1] \] is distinguished if and only if \[ B \xrightarrow{g} C \xrightarrow{h} A[1] \xrightarrow{-f[1]} B[1] \] is.
  3. Given a commutative diagram of distinguished triangles with vertical \(f\) and \(g\), there exists an \(h : C \to C'\) completing the diagram. Note: this \(h\) may be non-unique.
  4. The octahedron axiom.

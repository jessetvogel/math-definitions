The implicit function theorem states that if \(f : \RR^{n + m} \to \RR^m\) is a continuously differentiable function, and \(p \in \RR^{n + m}\) a point with \(f(p) = 0\) such that the Jacobian matrix \(J_f = \left(\frac{\partial f_i}{\partial x_j}\right)_{i,j = n + 1}^{n + m}\) is invertible at \(p\), then there exists an open set \(U \subset \RR^n\) containing \((p_1, \ldots, p_n)\) and a continuously differentiable function \(g : U \to \RR^m\) such that \(g(p_1, \ldots, p_n) = (p_{n + 1}, \ldots, p_{m + n})\) and \(f(y, g(y)) = 0\) for all \(y \in U\).

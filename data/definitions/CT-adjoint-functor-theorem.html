Let \(\mathcal{D}\) be a locally small and complete category. The adjoint functor theorem states that a functor \(G : \mathcal{D} \to \mathcal{C}\) has a left adjoint if and only if it preserves all limits and satisfies the solution set condition: for every object \(C\) of \(\mathcal{C}\), there exist objects \(D_i\) in \(\mathcal{D}\) and morphisms \(f_i : C \to G(D_i)\) in \(\mathcal{C}\), indexed by a set \(I\), such that every morphism \(h : C \to G(D)\) factors as \(h = G(t) \circ f_i\) for some \(t : D_i \to D\).

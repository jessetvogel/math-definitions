Let \((\mathcal{C}, \otimes, \textbf{1})\) be a monoidal category. An object \(X\) of \(\mathcal{C}\) is left dualizable if there exist an object \(Y\), called a left dual of \(X\), and morphisms \(\varepsilon : Y \otimes X \to \textbf{1}\) (evaluation) and \(\eta : \textbf{1} \to X \otimes Y\) (coevaluation) such that \[ X \xrightarrow{\eta \otimes \id_X} (X \otimes Y) \otimes X \xrightarrow{\alpha_{X, Y, X}^{-1}} X \otimes (Y \otimes X) \xrightarrow{\id_X \otimes \varepsilon} X \] and \[ Y \xrightarrow{\eta \otimes \id_Y} Y \otimes (X \otimes Y) \xrightarrow{\alpha_{Y, X, Y}} (Y \otimes X) \otimes Y \xrightarrow{\varepsilon \otimes \id_Y} Y \] are equal to \(\id_X\) and \(\id_Y\), respectively.
Similarly, one defines right dualizable objects by switching the roles of \(X\) and \(Y\). An object of \(\mathcal{C}\) is dualizable if it is both left and right dualizable.

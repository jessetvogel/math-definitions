A real symmetric-matrix \(M\) is
  • positive-definite if \(x^T M x > 0\) for all non-zero vectors \(x\),
  • negative-definite if \(x^T M x < 0\) for all non-zero vectors \(x\),
  • positive semi-definite if \(x^T M x \ge 0\) for all vectors \(x\),
  • negative semi-definite if \(x^T M x \le 0\) for all vectors \(x\).
Similarly, a Hermitian matrix \(H\) is
  • positive-definite if \(x^* M x > 0\) for all non-zero vectors \(x\),
  • negative-definite if \(x^* M x < 0\) for all non-zero vectors \(x\),
  • positive semi-definite if \(x^* M x \ge 0\) for all vectors \(x\),
  • negative semi-definite if \(x^* M x \le 0\) for all vectors \(x\).

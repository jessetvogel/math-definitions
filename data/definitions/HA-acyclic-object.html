Let \(F : \mathcal{A} \to \mathcal{B}\) be a left exact functor between abelian categories. An object \(A\) of \(\mathcal{A}\) is called acyclic with respect to \(F\), or \(F\)-acyclic, if \[ R^i F(A) = 0 \quad \textup{for all } i > 0, \] where \(R^i F\) are the right derived functors of \(F\).
Similarly, if \(F : \mathcal{A} \to \mathcal{B}\) is a right exact functor, an object \(A\) of \(\mathcal{A}\) is \(F\)-acyclic if \(L_i F(A) = 0\) for all \(i > 0\), where \(L_i F\) denote the left derived functors of \(F\).

A partial order on a set \(P\) is a binary relation \(\le\) on \(P\) such that
  • (reflexivity) \(x \le x\) for all \(x \in P\),
  • (anti-symmetry) if \(x \le y\) and \(y \le x\) then \(x = y\) for all \(x, y \in P\),
  • (transitivity) if \(x \le y\) and \(y \le z\) then \(x \le z\) for all \(x, y, z \in P\).
A partially ordered set is a set \(P\) together with a partial order \(\le\).

Let \(\mathcal{A}\) be an abelian category. An object \(I\) of \(\mathcal{A}\) is called injective if the functor \[ \Hom_\mathcal{A}(-, I) : \mathcal{A}^\textup{op} \to \textbf{Ab} \] is exact. Since it is already left exact for any object \(I\), this is equivalent to the condition that for all monomorphisms \(f : M \to N\) and morphisms \(g : M \to I\) there exists a morphism \(h : N \to I\) such that \(hf = g\).

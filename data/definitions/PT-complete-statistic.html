Let \(X_1, \ldots, X_n\) be a random sample from a population \(F_\theta\) depending on some parameter \(\theta\). A statistic \(T(X_1, \ldots, X_n) : \Omega \to (E, \mathcal{E})\) is complete for \(\theta\) if, for all measurable functions \(g : E \to \RR\), \[ \EE_\theta(g(T)) = 0 \textup{ for all } \theta \textup{ implies } \PP_\theta(g(T) = 0) = 1 \textup{ for all } \theta . \]

A semiring is a set \(R\) with two binary operations \(+ : R \times R \to R\), called addition, satisfying
  • (associative) \((a + b) + c = a + (b + c)\) for all \(a, b, c \in R\),
  • (commutative) \(a + b = b + a\) for all \(a, b \in R\),
  • (zero element) there exists an element \(0 \in R\) such that \(a + 0 = 0 + a = a\) for all \(a \in R\),
and \(\cdot : R \times R \to R\), called multiplication, satisfying
  • (associative) \((a \cdot b) \cdot c = a \cdot (b \cdot b)\) for all \(a, b, c \in R\),
  • (unit element) there exists an element \(1 \in R\) such that \(a \cdot 1 = 1 \cdot a = a\) for all \(a \in R\),
  • (distributive) \(a \cdot (b + c) = (a \cdot b) + (a \cdot c)\) and \((a + b) \cdot c = (a \cdot c) + (b \cdot c)\) for all \(a, b, c \in R\),
  • (zero element) \(0 \cdot a = a \cdot 0 = 0\) for all \(a \in R\).

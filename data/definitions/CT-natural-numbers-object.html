Let \(\mathcal{C}\) be a category with a terminal object \(1\). A natural numbers object in \(\mathcal{C}\) is a triple \((N, 0, S)\), where \(N\) is an object of \(\mathcal{C}\) and \(1 \xrightarrow{0} N\) and \(N \xrightarrow{S} N\) morphisms, such that for any other such triple \((N', 0', S')\), there is a unique map \(\phi : N \to N'\) for which commutes.

Let \(G = \textup{GL}_n(\CC)\) with \(B \subset G\) the subgroup of upper triangular matrices, \(T \subset G\) the subgroup of diagonal matrices, and \(W\) the symmetric group \(S_n\). The Bruhat decomposition says that for any \(A \in G\), we can write \(A = U P V\) with \(U, V \in B\) and \(P\) a permutation matrix. Rewriting as \(P = U^{-1} A V^{-1}\), this says any invertible matrix \(A\) can be brought into a permutation matrix via row and column operations, where we can only add row \(i\) to row \(j\) if \(i > j\), and column \(i\) to column \(j\) if \(i < j\).

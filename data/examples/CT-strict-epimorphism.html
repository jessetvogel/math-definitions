Any strict epimorphism \(f : X \to Y\) is an epimorphism. Suppose that \(\alpha, \beta : Y \to Z\) are morphisms with \(\alpha f = \beta f\). Then pick \(g = \alpha f = \beta f\) and note that \(fx = fy\) implies \(gx = gy\) for any \(x, y : W \to X\). Hence there is a unique morphism \(h : Y \to Z\) with \(g = hf\). Since \(\alpha\) and \(\beta\) both satisfy this property, it follows that \(\alpha = \beta\).
In Set, any epimorphism (surjective map) is a strict epimorphism, this is easy to check. However, in the category Top of topological spaces this is not the case. Consider \(X = [0, 1) \sqcup [1, 2]\) and \(Y = [0, 2]\) and the epimorphism (continuous surjective map) \(f : X \to Y, x \mapsto x\). Whenever two maps \(k, \ell : W \to X\) satisfy \(fk = f\ell\), it follows that \(k = \ell\) as this can be checked on the level of sets. However, there does not exist a continuous map \(h : Y \to X\) with \(\id_X = hf\).

Let \(G\) be a finite group, with irreducible representations \(\rho_1, \ldots, \rho_n\). Then the regular representation \(\rho\) decomposes as \[ \rho = \bigoplus_{i = 1}^{n} \rho_i^{\oplus \dim \rho_i} . \] Namely, if we take the elements of \(G\) as a basis for \(C(G)\), all \(\rho(g)\) are permutation matrices. In particular, it is easy to see that \(\chi(g) := \tr(\rho(g))\) equals the number of elements that are fixed by \(g\). Hence, \(\chi(1) = |G|\) and \(\chi(g) = 0\) if \(g \ne 1\). Using character theory, we find that \[ \rho = \bigoplus_{i = 1}^{n} \rho_i^{\oplus a_i} \quad \textup{ with } \quad a_i = \langle \chi, \chi_i \rangle = \frac{1}{|G|} \sum_{g \in G} \chi(g) \overline{\chi_i(g)} = \chi_i(1) = \dim \rho_i . \] In particular, we obtain \[ |G| = \chi(1) = \sum_{i = 1}^{n} (\dim \rho_i)^2 . \]

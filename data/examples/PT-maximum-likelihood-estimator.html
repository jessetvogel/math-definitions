Let \(X_1, \ldots, X_n\) be a random sample from a Bernoulli distribution with parameter \(0 \le p \le 1\). The likelihood function is \[ L(p; x_1, \ldots, x_n) = \prod_{i = 1}^{n} p^{x_i} (1 - p)^{n - x_i} = p^y (1 - p)^{n - y} \] where \(y = x_1 + \cdots + x_n\). A trick for finding the maximum of \(L(p; x_1, \ldots, x_n)\) is to maximize the log likelihood function \[ \log L(p; x_1, \ldots, x_n) = y \log p + (n - y) \log (1 - p) \] instead. One easily checks that a maximum is obtained for \(\hat{p} = y / n\), which is therefore the MLE for \(p\).

Let \(X = \AA^1_\CC\) and consider \(\alpha : J_\infty(X) \to \ZZ \cup \{ \infty \}\) given by \(\alpha(\gamma) = \operatorname{ord}_t(\gamma)\). Then \[ \pi_m(\alpha^{-1}(k)) = \left\{ \gamma \in \CC[t] / (t^{m + 1}) \mid \gamma = a_k t^k + \cdots + a_m t^m \textup{ with } a_k \ne 0 \right\} \] for any \(m \ge k\), and so \(\mu(\alpha^{-1}(k)) = (\mathbb{L} - 1) / \mathbb{L}^{k + 1}\). Therefore, \[ \int_{J_\infty(X)} \mathbb{L}^{-\alpha} d \mu = \sum_{k \ge 0} (\mathbb{L} - 1) \mathbb{L}^{-(k + 1)} \mathbb{L}^{-k} = \frac{\mathbb{L}}{\mathbb{L} + 1} . \]
Let \(X \subset \AA^2_\CC\) be the closed subvariety given by the equation \(xy = 0\). The formal arc space of \(X\) is given by \[ J_\infty(X) = \{ (\gamma_1, \gamma_2) \in \CC \llbracket t \rrbracket^2 \mid \gamma_1 \gamma_2 = 0 \} . \] Note that \(\gamma_1 \gamma_2 = 0\) implies \(\gamma_1 = 0\) or \(\gamma_2 = 0\). Therefore, the image of \(J_\infty(X)\) under \(\pi_m\) is not equal to \(J_m(X)\) but rather is given by \[ \pi_m(J_\infty(X)) = \{ (\gamma_1, \gamma_2) \in (\CC[t] / (t^{m + 1})^2 \mid \gamma_1 = 0 \textup{ or } \gamma_2 = 0 \} , \] whose virtual class is \([\pi_m(J_\infty(X))] = 2 \mathbb{L}^{m + 1} - 1\). It follows, for \(\alpha = 0\), that \[ \int_{J_\infty(X)} \mathbb{L}^{-\alpha} d \mu = \mu(J_\infty(X)) = \lim_{m \to \infty} [\pi_m(J_\infty(X))] / \mathbb{L}^{m + 1} = 2 . \]

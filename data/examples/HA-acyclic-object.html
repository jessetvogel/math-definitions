Given a left (resp. right) exact functor \(F : \mathcal{A} \to \mathcal{B}\) and an object \(A\) of \(\mathcal{A}\), an acyclic resolution of \(A\) is a resolution \(0 \to A \to I^0 \to I^1 \to \cdots\) (resp. \(\cdots \to P^1 \to P^0 \to A \to 0\)) of \(A\) where the \(I^i\) (resp. \(P^i\)) are \(F\)-acyclic objects. The left (resp. right) derived functors can be computed as \[ R^i F(A) = H^i(F(I^\bdot)) \qquad \left(\textup{resp. } L_i F(A) = H^i(F(P^\bdot)) \right) . \]
  • Given a commutative ring \(R\), the objects which are acyclic with respect to all functors \((-) \otimes_R M\), are the flat modules over \(R\).
  • The objects which are acyclic with respect to all functors \(\Hom_\mathcal{A}(A, -)\), are the injective objects. Moreover, an injective object is acyclic with respect to any left exact functor \(F\).
  • The objects which are acyclic with respect to all functors \(\Hom_\mathcal{A}(-, A)\), are the projective objects. Moreover, a projective object is acyclic with respect to any right exact functor \(F\).

Consider \(X = \PP^1\) over \(\FF_q\). Then \[ H^i(\overline{X}, \QQ_\ell) = \left\{ \begin{array}{cl} \QQ_\ell & \textup{ if } i = 0, \\ 0 & \textup{ if } i = 1, \\ \QQ_\ell(-1) & \textup{ if } i = 2, \\ \end{array} \right. \] and \[ \tr(F, H^i(\overline{X}, \QQ_\ell)) = \left\{ \begin{array}{cl} 1 & \textup{ if } i = 0, \\ 0 & \textup{ if } i = 1, \\ q & \textup{ if } i = 2, \\ \end{array} \right. \] so the Grothendieck trace formula yields \[ \# \PP^1(\FF_q) = 1 + q , \] as expected.

• For \(\lambda = (n)\), corresponding to the trivial representation, we have \(\ell_1 = n\). Note that \(1 \cdot \prod_{j = 1}^{n} (x_1^j)^{i_j} = x_1^n\) as \(\sum_{j = 1}^{n} j \cdot i_j = n\), so \(\chi_\lambda(C_i) = 1\) as expected.
  • For \(\lambda = (n - 1, 1)\), corresponding to the standard representation, we have \(\ell_1 = n\) and \(\ell_2 = 1\). The coefficient of \(x_1^n x_2\) in \((x_1 - x_2) \prod_{j = 1}^{n} (x_1^j + x_2^j)^{i_j}\) is \(i_1 - 1\), so \(\chi_\lambda(C_i) = i_1 - 1\).
  • For \(\lambda = (1, 1, \ldots, 1)\), corresponding to the alternating representation, we have \(\ell_j = n + 1 - j\). The coefficient of \(x_1^{\ell_1} \cdots x_n^{\ell_n}\) in \(\prod_{1 \le i < j \le n} (x_i - x_j) \prod_{j = 1}^{n} (x_1^j + \cdots x_n^j)^{i_j}\) should be \((-1)^{(i_2 + i_4 + \cdots)}\) as \(\chi_\lambda(C_i) = (-1)^{(i_2 + i_4 + \cdots)}\).

• Let \(\mathcal{A}\) be any \(\sigma\)-algebra on a set \(\Omega\), and take \(\omega \in \Omega\). The Dirac measure is given by \[ \delta_\omega(A) = \left\{ \begin{array}{cl} 1 & \textup{ if } \omega \in A, \\ 0 & \textup{ if } \omega \not\in A . \end{array} \right. \]
  • Let \(\mathcal{A} = P(\Omega)\) for a set \(\Omega\). The counting measure on \(\mathcal{A}\) is given by \(\mu(A) = |A|\).

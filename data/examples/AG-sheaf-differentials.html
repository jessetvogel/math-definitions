Let \(f : X \to Y\) be locally of finite type. If \(f\) is unramified then \(\Omega_{X/Y} = 0\). Namely, since \(\Omega_{X/Y}\) behaves well with respect to base change, this can be checked locally, that is, we can assume \(Y = \Spec B\) and \(X = \Spec A\) with \(A \to B\) a local homomorphism of local rings. Using Nakayamaâ€™s lemma this reduces to the case where \(A\) and \(B\) are fields. Being unramified means that \(A \subset B\) is a separable field extension, and then it is clear that \(\Omega_{B/A} = 0\).
Conversely, if \(\Omega_{X/Y} = 0\) then \(f\) is unramified.

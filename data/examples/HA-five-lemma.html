• Take any \(\gamma \in \ker c\). Then \(dh(\gamma) = h'c(\gamma) = 0\), so \(h(\gamma) = 0\) by injectivity of \(d\), and thus we can write \(\gamma = g(\beta)\) for some \(\beta \in B\). Now, \(g'b(\beta) = cg(\beta) = c(\gamma) = 0\), so \(b(\beta) \in \ker g' = \im f'\), and thus we can write \(b(\beta) = f'(\alpha')\) for some \(\alpha' \in A'\). Since \(a\) is surjective, we have \(\alpha' = a(\alpha)\) for some \(\alpha \in A\). Now \(b(f(\alpha)) = f'a(\alpha) = f(\alpha') = b(\beta)\), so \(f(\alpha) = \beta\) by injectivity of \(b\). Therefore, \(\gamma = gf(\alpha) = 0\), and thus \(c\) is injective.

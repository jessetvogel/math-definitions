Consider \(M = \RR^n\) with the Euclidean metric, and \(E = M \times \CC\) the trivial complex vector bundle. Then \(D = \sum_{j = 1}^{n} i \partial_{x_j}\) is a Dirac operator for \(E\), because \[ \sigma_D(dx_j) = [D, x_j] = [i \partial_{x_j}, x_j] = i \] so \(\sigma_D(dx_j)^2 = -1 = - g(dx_j, dx_j)\) for all \(k = 1, \ldots, n\).

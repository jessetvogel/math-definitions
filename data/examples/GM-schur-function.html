For \(n = 3\), we have
  • \(s_{(0, 0, 0)} = 1\),
  • \(s_{(1, 0, 0)} = x_1 + x_2 + x_3\),
  • \(s_{(1, 1, 0)} = x_1 x_2 + x_1 x_3 + x_2 x_3\),
  • \(s_{(1, 1, 1)} = x_1 x_2 x_3\),
  • \(s_{(2, 0, 0)} = x_1^2 + x_1 x_2 + x_1 x_3 + x_2^2 + x_2 x_3 + x_3^2\),
  • \(s_{(2, 1, 0)} = (x_1 + x_2) (x_1 + x_3) (x_2 + x_3)\).
For any \(n \ge 1\) and \(\lambda = (1, 0, 0, \ldots)\), we have \(s_\lambda = \sum_{i = 1}^{n} x_i\), so we obtain the Schur function \(s_\lambda = \sum_{i = 1}^{\infty} x_i \in \Lambda\).

Suppose \(F_\theta\) is a Bernoulli distribution with parameter \(0 < p = \theta < 1\), and let \(T(X_1, \ldots, X_n) = \sum_i X_i\). Then \(T\) follows a binomial distribution with parameters \(n\) and \(p\), and we claim that \(T\) is a complete statistic for \(\theta = p\). Namely, if \(g\) is a measurable function such that \(\EE_\theta(g(T)) = 0\) for all \(\theta\), then \[ \EE_\theta(g(T)) = \sum_{t = 0}^{n} g(t) \binom{n}{t} p^t (1 - p)^{n - t} = (1 - p)^n \sum_{t = 0}^{n} g(t) \binom{n}{t} \left(\frac{p}{1 - p}\right)^t = 0 \] for all \(0 < p < 1\). But this is a polynomial in \(r = p / (1 - p) > 0\), so it can only be identically zero if all its coefficients are zero, that is, \(g(t) = 0\) for all \(t\). Hence, \(g\) is identically zero and \(T\) is a complete statistic for \(\theta = p\).

Let \(X\) be a topological space. The singular set of a topological space is the simplicial set \(\mathcal{S}(X)\) given by \[ \mathcal{S}(X)_n = \{ \text{cont. maps } \Delta^n \to X \} . \] Every \([n] \to [m]\) induces a map \(\Delta^n \to \Delta^m\) by labelling the vertices, which in turn induces a map \(\mathcal{S}(X)_m \to \mathcal{S}(X)_m\). Also see singular homology.

Given a complex semisimple Lie algebra \(\mathfrak{g}\) and a Cartan subalgebra \(\mathfrak{h} \subset \mathfrak{g}\), one can construct a root system \(\Phi\) in \(\mathfrak{h}^*\), where a root is an element \(\alpha \in \mathfrak{h}^*\) such that \[ \mathfrak{g}_\alpha = \{ x \in \mathfrak{g} : [h, x] = \alpha(h) x \textup{ for all } h \in \mathfrak{h} \} \] is non-empty.

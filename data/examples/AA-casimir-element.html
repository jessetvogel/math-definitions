Consider \(\mathfrak{g} = \mathfrak{sl}_2(\CC)\) with the usual basis \[ X = \begin{pmatrix} 0 & 1 \\ 0 & 0 \end{pmatrix}, \quad Y = \begin{pmatrix} 0 & 0 \\ -1 & 0 \end{pmatrix} \quad \textup{ and } \quad H = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix} . \] With respect to the Killing form \(\kappa_\mathfrak{g}\), we have \[ \kappa_\mathfrak{g}(X, X) = 0, \quad \kappa_\mathfrak{g}(X, Y) = 4, \quad \kappa_\mathfrak{g}(X, H) = 0, \] \[ \kappa_\mathfrak{g}(Y, Y) = 0, \quad \kappa_\mathfrak{g}(Y, H) = 0, \quad \kappa_\mathfrak{g}(H, H) = 8, \] so it follows that \(X^* = \frac{1}{4} Y\), \(Y^* = \frac{1}{4} X\) and \(H^* = \frac{1}{8} H\). Hence, the Casimir element of \(\mathfrak{sl}_2(\CC)\) is \[ \frac{1}{4} XY + \frac{1}{4} YX + \frac{1}{8} H^2 \in U(\mathfrak{sl}_2(\CC)) . \] Since \(\Omega\) is central, it acts by a scalar on every representation of \(\mathfrak{g}\). In this case, for a representation \(V\) of highest weight \(m\), the Casimir element \(\Omega\) acts by the scalar \(\frac{1}{8} m(m + 2)\).

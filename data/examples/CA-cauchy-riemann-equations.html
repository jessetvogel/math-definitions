Consider the function \(f(z) = z^2\), which can be written as \(f(x + iy) = u(x, y) + iv(x, y)\) with \(u(x, y) = x^2 - y^2\) and \(v(x, y) = 2xy\). Then we see that \[ \frac{\partial u}{\partial x} = 2x = \frac{\partial v}{\partial y} \quad \textup{ and } \quad \frac{\partial u}{\partial y} = -2y = -\frac{\partial v}{\partial x} , \] so \(f\) is holomorphic.
